# NASSCOM AI Workshop Learning Repository

## Overview

This repository contains learning materials, exercises, notes, and hands-on activities completed during the **AI Workshop organized by NASSCOM** at **VEDIC Vishnu Universal Learning**, Ghazib Nagar, Hyderabad.

The objective of this repository is to systematically document daily learning outcomes, practical exercises, project activities, and key insights gained throughout the workshop.

## Workshop Details

* **Workshop:** NASSCOM AI Workshop
* **Organizer:** NASSCOM
* **Venue:** VEDIC Vishnu Universal Learning, Ghazib Nagar, Hyderabad
* **Participant:** Dr B. Murali Krishna
* **Representing:** Shri Vishnu Engineering College for Women (SVECW), Bhimavaram
* **Focus Areas:** Artificial Intelligence, Data Science, Machine Learning, Generative AI, Agentic AI, and Hands-on AI Applications

Throughout this workshop, I aim to:
